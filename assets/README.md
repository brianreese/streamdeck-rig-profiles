# Assets

Place Stream Deck icon files here. The following filenames are referenced in `manifest.json`:

- `plugin-icon.png` (128×128) — plugin icon shown in the Stream Deck store/gallery
- `plugin-icon@2x.png` (256×256) — HiDPI version
- `action-icon.png` (72×72) — default button image for the Profile Toggle action
- `action-icon@2x.png` (144×144) — HiDPI version
